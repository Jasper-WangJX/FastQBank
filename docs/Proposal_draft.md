# AI-ASSISTED QUESTION BANK
## 功能
### 1. 添加题目
- 通过屏幕识别读取文字并自动分离题目和选项，并提供快捷工具让用户选择正确选项和易错选项
- 用户可以预先设置标签分类，ai读题后自动从已存在的标签中推荐匹配标签便于用户分类
- 在软件内部也可手动输入题目

### 2. 数据存储
- 数据可以本地存储也可以保存到云端，实现不同软件间互通
- 同时可以将题目打包导出，并在其他平台上导入

### 3. 多平台实现
- 软件应在 windows/MacOS/Android/IOS/Website 平台均实现

### 4. 快速过题/AI出题
- 软件内查看题目时可一键选择是否显示正确选项/是否乱序
- 可从题库选择题目进行flashcards式快速过题
- 在上面的过题功能的基础上加入ai出题的功能，根据被选择的题目知识点，使用ai生成类似题目，用户可选择‘仅导入题目’，‘仅ai生成题目’，‘混合’模式
